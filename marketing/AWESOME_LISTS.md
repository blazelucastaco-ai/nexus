## How to submit (the mechanics)

For each list above:

1. Open the repo README on GitHub
2. Click the pencil icon (top-right) — GitHub auto-forks for you
3. Find the right section alphabetically
4. Add your entry in sorted position
5. Commit with message: `Add NEXUS`
6. GitHub offers "Propose changes" → creates the PR automatically

Keep PR descriptions short:

> Adds NEXUS, an open-source personal AI agent for macOS. Telegram-first, four-layer SQLite memory, 22 MB idle, built on Anthropic Claude. MIT licensed.
>
> Repo: https://github.com/blazelucastaco-ai/nexus
> Site: https://blazelucastaco-ai.github.io/nexus/

---

## Don't

- **Don't spam-submit to 50 awesome-lists in one day.** GitHub flags the behaviour and maintainers reject on sight. Three or four high-quality, well-fitted submissions in a week beats thirty drive-by PRs.
- **Don't submit to lists that are clearly dead** (no merged PR in 6+ months). Check the "Pulse" tab.
- **Don't add NEXUS to a category it doesn't clearly fit.** "AI agent" on awesome-macos is fine; "image generation" is not.
