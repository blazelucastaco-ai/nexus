What is NEXUS

Not a chatbot.
An agent OS.

NEXUS runs persistently on your Mac, thinks between messages, dreams at night, and controls your entire computer. It's the difference between a calculator and a co-worker.

🧠

Persistent Memory

Four-layer memory architecture — episodic, semantic, procedural, and working — with nightly dream cycles that consolidate and reflect on the day's context.

💬

Telegram-First Interface

Control NEXUS from anywhere in the world via Telegram. 11 slash commands, media support, inline keyboards, and live status updates.

🌐

Full Browser Control

16 browser tools including screenshot capture, hover, scroll, click, form fill, and cookie management — powered by a native Chrome extension via WebSocket bridge.

⚡

Autonomous Execution

Write and run code, manage files, execute shell commands, install packages, and build entire projects from a single Telegram message.

❤️

Emotional Intelligence

Real emotional state that drifts based on interactions. NEXUS tracks satisfaction, frustration, and engagement — adapting its personality over time.

🔒

Local-First Privacy

Everything runs on your Mac. SQLite for memory, JSONL for session history, and local filesystem. No cloud sync, no data leaving your machine.

How It Works

A 13-step message
pipeline.

Every message passes through a fully coordinated processing chain before NEXUS replies. Nothing is left to chance.

1
Message received via Telegram
Raw message arrives through Grammy bot gateway with metadata.
2
Classify message type
Zero-LLM pattern matching: chat, task, ultra, coordinator, or probe.
3
Inner monologue
For complex tasks: an internal reasoning pass before responding.
4
Memory synthesis
Retrieves and synthesizes relevant memories for context injection.
5
Tool execution
Browser, code runner, file manager, web search — in parallel.
6
Self-evaluation
Completeness check triggers self-correction if needed.
7
Memory storage + learning
Response and context persisted; preferences and patterns updated.

Installation

Up in five minutes.

NEXUS is a TypeScript project that installs like any npm package. The interactive setup wizard handles everything else.

1

Clone the repository

Clone from GitHub and enter the project directory.

2

Install dependencies

Uses npm. Requires Node.js 20+ and a Mac (macOS 12+).

3

Run the setup wizard

Interactive CLI that configures your Anthropic API key, Telegram bot token, personality, and Chrome extension.

4

Build and start

TypeScript compilation takes ~10 seconds. NEXUS starts and sends you a "Good morning" briefing.

5

Message NEXUS on Telegram

That's it. Try /status to confirm everything is running.

bash 
# 1. Clone
git clone https://github.com/blazelucastaco-ai/nexus
cd nexus

# 2. Install
npm install

# 3. Setup wizard (interactive)
npm run setup

# 4. Build + start
npm run build
npm start

# 5. Re-install Chrome extension anytime
nexus extension
