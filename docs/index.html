What is NEXUS

Not an assistant.
A resident AI.

NEXUS is a fully autonomous AI agent built in TypeScript that lives persistently on your Mac. It connects via Telegram, controls your system, and accumulates memory across every session — developing a genuine understanding of who you are and how you work.

🧠 The core idea

Most AI tools reset every conversation. NEXUS doesn't. It maintains four layers of persistent memory — episodic (what happened), semantic (what it knows), procedural (how things work), and working memory for the current task. Every night it runs a dream cycle: consolidating memories, decaying stale ones, generating reflections, and storing new ideas while you sleep. When you message NEXUS in the morning, it already knows what's on your plate.

📱 Telegram-first

Your entire interface is Telegram. Message from your phone anywhere in the world via Tailscale and NEXUS responds like a colleague who's been at your desk all day. No web app to open, no context to re-establish.

💻 True Mac control

NEXUS can run terminal commands, write and execute code, read and write files, automate Chrome via the NEXUS Bridge extension, take screenshots, query the macOS accessibility tree, and control apps via AppleScript.

🎭 Genuine personality

Configurable humor, sarcasm, and emotional state. NEXUS tracks whether its jokes land (literally), dampens humor when they don't, and notices when you're frustrated. Its style adapts to your mood and the conversation's depth.

🔒 Security-first

Every terminal command passes through a four-tier approval policy: SAFE, MODERATE, DANGEROUS, and BLOCKED. DANGEROUS commands require explicit confirmation. Injection attempts are caught before they reach the LLM. Your system stays yours.

Feature Set

Everything an AI agent should do.

Fourteen production-grade subsystems, all wired together and running continuously.

🌙

Dream Cycle

Every 6 hours, NEXUS enters a dream cycle: consolidates high-access episodic memories into semantic facts, decays stale entries, runs contradiction resolution, generates reflections, produces actionable ideas, and journals everything to long-term memory.

Memory · Scheduled
🧩

4-Layer Memory System

Episodic (events), semantic (facts), procedural (skills), and short-term buffer. Full recall uses embedding-based semantic search with importance scoring, access-count weighting, and keyword fallback. Nothing is ever truly forgotten unless it should be.

Core · Persistent
🧬

Memory Synthesizer

Before each response, a fast LLM call synthesizes raw memory fragments into a coherent paragraph. Results are cached for 60 seconds per query hash. Skips synthesis when fewer than 3 memories are available to avoid wasted tokens.

Cognition · Optimized
💭

Inner Monologue

Optionally generate a private 2–4 sentence thought before responding — surfacing relevant memories, emotional context, and reasoning intent. Automatically skips trivial or short messages to keep latency low.

Cognition · Optional
🌐

Chrome Browser Control

Full browser automation via the NEXUS Bridge Chrome extension — navigate, extract content, fill forms, click, scroll, wait for elements, dismiss cookie banners, suppress dialogs, manage tabs, and take screenshots. No Playwright scripting needed.

Tools · Browser
🎯

Goal Tracker

Automatically detects goal statements in your messages ("I want to…", "my goal is…", "by next week…") and stores them as high-importance memories. Goals appear in every system prompt. Stale goals (30+ days inactive) are pruned by keyword overlap analysis.

Learning · Proactive
📊

Pattern Recognition

Tracks temporal patterns (time-of-day clustering, weekend activity), sequence patterns (A→B→C behavioral chains), and preference patterns (dominant choices per category). Persists detected patterns to the database for cross-session continuity.

Learning · Analytics
🎭

Humor Engine

Decides whether to inject humor based on topic seriousness, conversation length, rapport score, and frequency caps. Tracks whether jokes land (lol, 😂, "good one") vs miss. Dampens humor up to 50% when the hit rate drops below 20% over 20+ attempts.

Personality · Adaptive
⏰

Task Scheduler

Cron-based scheduled tasks with SQLite persistence — reminders and recurring automations survive restarts. Tracks last exit code, execution duration, and consecutive failure count. Auto-disables tasks after 5 consecutive failures.

Automation · Persistent
☀️

Daily Briefing

Each morning at a configurable hour, NEXUS sends a personalized briefing: last dream summary, scheduled tasks for today, top detected pattern, recurring mistakes to watch for, and an LLM-generated thought based on recent memory context.

Productivity · Daily
🔍

Self-Evaluator

After task-type responses, asynchronously evaluates whether the answer fully addressed the question. Gaps are stored as episodic reflections and routed to the mistake tracker. Only fires for task messages with responses over 200 characters.

Quality · Background
🛡️

Injection Guard

Every message passes through a multi-layer injection detection pipeline before reaching the LLM. Detects classic prompt injection patterns, role override attempts, instruction smuggling via code/links/images, and Unicode obfuscation techniques.

Security · Always-on
📈

Mistake Tracker

Records categorized mistakes with root causes and prevention strategies. Detects similar past mistakes via Jaccard keyword overlap (≥30%). Automatically escalates severity every 2 recurrences (minor → moderate → major → critical).

Learning · Self-improving
🔭

Proactive Monitor

Background engine monitors ports and system health. Implements exponential backoff on alerts (5min → 30min → 2hr) to avoid notification spam. Sends Telegram alerts on state changes.

Monitoring · Background

How It Works

Every message is a 13-step pipeline.

From Telegram to response, each message flows through the full NEXUS orchestration stack — memory, cognition, tools, and personality all woven together.

1

Injection Guard

The raw message is scanned for injection patterns, role overrides, and Unicode tricks before anything else runs.

2

Memory Recall

Embeddings + keyword scoring retrieves the most relevant episodic and semantic memories. User facts and active goals are loaded.

3

Memory Synthesis

A fast LLM call synthesizes memory fragments into a coherent paragraph. Cached for 60 seconds per query hash to avoid redundant calls.

4

Goal & Intent Detection

Goal patterns are extracted and stored. The message is classified as task or chat, and an execution mode is assigned (standard/coordinator/ultra).

5

Task Planning (if needed)

Task messages get a structured plan with 2–15 steps. Ultra-mode tasks require a strong-model review pass before execution begins.

6

Inner Monologue (optional)

If think mode is enabled and the message is substantial, a private reasoning pass runs before the main response — surfacing context and setting intent.

7

LLM Response + Tool Execution

The orchestrator calls Claude with a full context window. Tool calls (terminal, browser, file, web) execute in a sandboxed approval loop.

8

Self-Evaluation (async)

After the response is returned, a background completeness check identifies gaps and routes them to the mistake tracker and episodic memory.

Installation

Up and running in minutes.

NEXUS runs as a persistent launchd service on macOS. All you need is Node.js 22+, an Anthropic API key, and a Telegram bot token.

1
Clone & install dependencies
# Requires Node.js 22+ and pnpm
git clone https://github.com/blazelucastaco-ai/nexus.git
cd nexus
npm install
2
Configure environment
# Create .env in the project root
ANTHROPIC_API_KEY="sk-ant-..."
TELEGRAM_BOT_TOKEN="12345:ABC..."
TELEGRAM_CHAT_ID="your_chat_id"
BRAVE_API_KEY="BSA..." # optional
3
Build the project
npm run build
 
# Output: dist/index.js, dist/cli.js
# Auto-copies to ~/.nexus/app/index.cjs
4
Run the setup wizard
npm run setup
 
# Interactive CLI wizard installs the
# launchd plist and starts the service
5
Verify NEXUS is running
# Check service status
launchctl list | grep nexus
 
# Or use the CLI
nexus status
 
# Send a test message on Telegram
# → Type "hello" and NEXUS should respond within 2 seconds
 
# To trigger a manual dream cycle
nexus dream
 
# To run tests (418 passing)
npm test
💡
Install the NEXUS Bridge Chrome Extension
For full browser automation, load the chrome-extension/ folder as an unpacked extension in Chrome's developer mode. Once connected, NEXUS gains 16 browser_* tools for navigating, extracting, clicking, and filling any website.
